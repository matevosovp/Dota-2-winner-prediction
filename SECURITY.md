# Security policy

## Данные и секреты

- Не коммитьте исходные CSV/JSONL, токены API, персональные данные игроков и локальные experiment artifacts.
- Получайте датасет только из доверенного источника и проверяйте условия его распространения.
- JSONL рассматривается как недоверенный ввод: пайплайн парсит его построчно и извлекает только allow-listed поля.

## Артефакты моделей

Репозиторий не публикует pickle/joblib-артефакты. Не загружайте такие файлы из непроверенных источников: десериализация может выполнить произвольный код.

## Уязвимости

Для приватного сообщения используйте GitHub Security Advisories репозитория. В публичном issue не размещайте секреты, приватные данные или готовый exploit.
