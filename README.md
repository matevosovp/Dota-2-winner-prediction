
# Dota 2: предсказание победы 

Проект бинарной классификации на табличных данных и логах матчей: предсказать вероятность победы команды **Radiant** по состоянию матча на фиксированный момент времени.

Что демонстрирует проект:
- аккуратный EDA и контроль качества данных
- feature engineering для табличных признаков и извлечение сигналов из JSONL логов матчей
- сравнение бейзлайнов и бустинга 
- корректная кросс-валидация без утечек
- подбор гиперпараметров (Optuna) и интерпретация важности признаков
- анализ ошибок и идеи улучшений

## Результат
- Метрика: ROC-AUC
- Лучший результат по CV: 0.84
